# Sonical Client
HTTP application built with Elm and Bootstrap v4.

## Dependencies
* Elm >= 0.18.x
* Node >= 7.5.0
* Npm >= 4.1.2

## Setup
`$ npm install`
`$ elm-package install`

## Develop
`$ npm start`

## Deploy
`$ npm run build`
